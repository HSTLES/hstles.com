Protection

Multi-layered security at every level

Data Sovereignty

Primary storage and processing in Sydney-based Australian data centres. Backups encrypted within Australian jurisdiction. Data never leaves Australian borders unless explicitly requested.

Encryption Standards

TLS 1.2/1.3 for all data in transit. AES-256 encryption for data at rest. All credentials and API keys stored in secure vaults, never in code.

Infrastructure Security

Firecracker microVMs for strong tenant isolation. Private networking with no direct database exposure. Ephemeral infrastructure to minimise attack surface.

Access Control

OIDC/OAuth2 authentication. Role-Based Access Control (RBAC) at every level. Principle of least privilege applied throughout.

Mesh Encryption

VL1 overlay with Noise Protocol encryption and mTLS for internal communications. Protocol break between HTTP and internal mesh acts as a natural firewall.

Vulnerability Management

Regular dependency scanning for known vulnerabilities. Automated security patching of the underlying platform. Secure development practices with code review.

Resilience

Backup & Disaster Recovery

Your data, protected against the unexpected. Automated daily backups with rolling 30-day retention (configurable per client). Regular tested restoration procedures and multiple availability zones within region.
